# News Article Analyzer

## Description
This Python application analyzes a news article and provides useful statistics about the text.

## Features
- Count occurrences of a specific word
- Identify the most common word
- Calculate average word length
- Count paragraphs
- Count sentences

## Files
- main.py
- text_analysis.py
- sample_article.txt

## How to Run

1. Open a terminal.
2. Navigate to the project folder.
3. Run:

```bash
python main.py
Choose option 1.
Enter a word to search for.
